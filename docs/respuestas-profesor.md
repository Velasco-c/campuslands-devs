# Rama respuestas-profesor

Esta rama contiene soluciones guía preparadas por el profesor para los ejercicios del nivel básico.

## Propósito

Servir como material de consulta después de que el estudiante haya intentado resolver el ejercicio por su cuenta.

## Uso recomendado

1. El estudiante trabaja primero desde `dev` en su propia rama.
2. Resuelve el ejercicio sin mirar esta rama.
3. Sube su solución y abre Pull Request.
4. Después de entregar, puede consultar `respuestas-profesor` para comparar enfoque, lógica, estructura y validación.

## Reglas

- Esta rama no es la rama de trabajo de los alumnos.
- Las soluciones no deben copiarse como entrega propia.
- Los archivos de esta rama sirven como guía, no como única respuesta posible.
- Si una solución de alumno funciona correctamente con otro enfoque, también puede ser válida.

## Ubicación de respuestas

Cada ejercicio tiene una respuesta guía en:

```text
basico/<area>/ejercicio-XX/resoluciones/solucion-profesor.md
```

Ejemplo:

```text
basico/logica/ejercicio-01/resoluciones/solucion-profesor.md
```
