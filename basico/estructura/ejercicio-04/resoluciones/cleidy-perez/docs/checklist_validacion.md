# Checklist de validación

## Datos

- [ ] Todos los autos poseen un ID único.
- [ ] Todos los registros tienen marca y año.
- [ ] El nombre de la imagen coincide con el JSON.

## Imágenes

- [ ] No existen imágenes duplicadas.
- [ ] Las imágenes siguen la convención establecida.
- [ ] Todas las imágenes están dentro de assets/cars.

## Código

- [ ] La lógica está separada de la visualización.
- [ ] No existen errores en consola.
- [ ] Se documentaron las funciones principales.
