# Componente: CardDestino (Tarjeta de Destino)

Este componente es una pieza de interfaz reutilizable diseñada para mostrar un resumen visual y comercial de los lugares disponibles en la plataforma de turismo.

## Estructura Visual Esperada (HTML)
```html
<div class="destination-card">
    <img src="ruta-imagen.jpg" alt="Nombre del Destino" class="destination-image" />
    <div class="destination-content">
        <span class="destination-badge">País</span>
        <h3 class="destination-title">Nombre del Lugar</h3>
        <p class="destination-price">Desde $XXX USD</p>
        <button class="btn-book">Reservar Ahora</button>
    </div>
</div>
