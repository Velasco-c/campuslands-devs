# Politicas Comerciales e Ingenieria de Menu

Este documento establece las reglas financieras obligatorias para la fijacion de precios dentro del menu urbano.

## 1. Estructura de Costos e Impuestos
- Todos los precios cargados en los archivos JSON correspondientes a las carpetas `menu/` deben incluir el 12% del Impuesto al Valor Agregado (IVA).
- No se permiten valores negativos ni precios en cero en ningun item del menu.

## 2. Reglas de Descuento para la Carpeta 'combos/'
- Un combo debe estar integrado obligatoriamente por un minimo de un item de `platos/` y un item de `bebidas/`.
- El precio final asentado en el archivo JSON del combo debe aplicar un descuento obligatorio de entre el 15% y el 20% en comparacion con la suma de los precios de los productos adquiridos de forma individual.
